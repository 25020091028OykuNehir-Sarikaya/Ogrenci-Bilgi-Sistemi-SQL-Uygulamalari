# Ogrenci-Bilgi-Sistemi-SQL-Uygulamalari
MSSQL ve MySQL Veritabanı Ödevi  Bu proje; i7-13620H işlemci, 32GB RAM ve RTX 5070 ekran kartlı yüksek performanslı sistemde, MSSQL Server ve XAMPP (MySQL) ortamları kullanılarak hazırlanmıştır. "Okul" veritabanı ve "Ogrenciler" tablosu üzerinde temel CRUD (SELECT, INSERT, UPDATE, DELETE) operasyonları her iki platformda ayrı ayrı test edilmiştir.
